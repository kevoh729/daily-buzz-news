# Daily Buzz News Blog

A simple HTML/CSS news website created as a first web development project.  
It features a header with logo, horizontal navigation, and news cards with images, shadows, and hover effects.

## Features

- Logo in the header and as favicon
- Horizontal navigation bar linking to multiple pages (Home, About, Contact)
- News card layout with images, titles, and descriptions
- Shadow and hover effects on news cards for visual depth
- Responsive design: cards wrap on smaller screens
- Dark-themed design with readable text and consistent card sizes
- Local images included for offline use

## How to Run

1. Download or clone the repository.
2. Ensure the `images` folder and CSS folder are included.
3. Open `index.html` in any web browser.
4. Navigation links allow moving between pages.
